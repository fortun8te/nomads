📌 How to Get These Into Figma

Step 1: Create Frames for Each Concept

In your Figma file, create 3 new frames named "Ad Concept 1", "Ad Concept 2", "Ad Concept 3". Set each to 1200px × 800px (desktop) + variants for mobile (375px × 812px).

Step 2: Design in Figma (or use Code Connect)

Option A: Manually recreate the designs in Figma based on the HTML layouts. Option B: Use Code Connect to link the Figma frames to the HTML code below.

Step 3: Add Code Connect Mapping

Install Code Connect, then add this configuration to link your Figma designs to the HTML code:

// In figma.config.json or your design system: { "concepts": [ { "id": "ad-concept-1", "name": "Static Hero Ad", "figmaComponentId": "YOUR_FIGMA_ID", "htmlPath": "/ad-concept-1.html", "description": "Bold hero with social proof metrics" }, { "id": "ad-concept-2", "name": "Proof Stack Ad", "figmaComponentId": "YOUR_FIGMA_ID", "htmlPath": "/ad-concept-2.html", "description": "Testimonials + stats + features" }, { "id": "ad-concept-3", "name": "Urgency Carousel Ad", "figmaComponentId": "YOUR_FIGMA_ID", "htmlPath": "/ad-concept-3.html", "description": "Interactive 3-step carousel" } ] }
Step 4: HTML Files Ready to Use

All HTML files are production-ready with:

  • Fully responsive CSS (mobile, tablet, desktop)
  • Conversion-optimized layout and UX
  • No external dependencies
  • Qwen-generated custom designs (not templates)
