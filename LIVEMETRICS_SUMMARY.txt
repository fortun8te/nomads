QUICK START (Copy-Paste)
═══════════════════════════════════════════════════════════════════════════════

1. Import:
   import { LiveMetricsPanel } from './components/LiveMetricsPanel';
   import { useMetrics } from './hooks/useMetrics';
   import { useCampaign } from './context/CampaignContext';

2. Use hook:
   const { currentCycle, isRunning } = useCampaign();
   const metricsInput = useMetrics(currentCycle, isRunning);

3. Render:
   {isRunning && <LiveMetricsPanel {...metricsInput} />}

4. Update Cycle during research:
   cycle.orchestrationData = { iteration: 8, maxIterations: 30, ... };
   cycle.watchdogState = { tokensUsed: 127000, ... };
   await saveCycle(cycle);

Done! Panel updates automatically. 🎉

DATA STRUCTURE
═══════════════════════════════════════════════════════════════════════════════

The panel reads from Cycle fields:

✓ cycle.orchestrationData
  • iteration: number
  • maxIterations: number
  • coveragePercent: number
  • coverageDimensions: string[]
  • coverageDimensionCounts: Record<string, number>

✓ cycle.watchdogState
  • tokensUsed: number
  • tokenBudget: number
  • iterationsRemaining: number
  • stagnationRounds: number
  • queryRepeatCount: Record<string, number>
  • shouldKill: boolean
  • killReason?: string

✓ cycle.stageMetrics
  • currentStage: string
  • elapsedMs: number
  • currentModel: string
  • thinkingTokens: number

✓ cycle.researchFindings
  • pagesScanned: number
  • urlsProcessed: number
  • keyFactsExtracted: number
  • visualFindings?.competitorVisuals: any[]

✓ cycle.activeResearchers
  • query: string
  • progress: number (0-100)
  • status: 'pending' | 'running' | 'complete' | 'error'
