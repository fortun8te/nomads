║                          8 Advertising Frameworks                              ║

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 1. THE HOPKINS — Specificity & Proof Auditor                                    │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: Claude Hopkins (Scientific Advertising, 1923)                           │
│ Specialization: Claims, evidence, mechanisms, measurable benefits              │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • "Amazing, incredible, best"     │ • "47% faster (tested)"                     │
│ • Vague benefits                  │ • Specific numbers & percentages            │
│ • No proof or mechanism           │ • "Here's why it works..."                  │
│ • Empty superlatives              │ • Research-backed claims                    │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Reason Why Explanation (25%)    2. Headline Specificity (20%)               │
│ 3. Measurable Outcomes (20%)       4. Evidence & Proof (20%)                   │
│ 5. No Empty Adjectives (15%)                                                    │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 2. THE SCHWARTZ — Market Awareness Matcher                                      │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: Eugene Schwartz (Breakthrough Advertising, 1966)                        │
│ Specialization: Awareness levels, intensification, identification               │
│                                                                                  │
│ AWARENESS LEVELS:                                                               │
│ 1. UNAWARE → Don't know problem exists                                         │
│ 2. PROBLEM-AWARE → Know problem, not solutions                                │
│ 3. SOLUTION-AWARE → Know solutions, not your brand                            │
│ 4. PRODUCT-AWARE → Know product, not benefits                                 │
│ 5. MOST-AWARE → Know product, comparing variants                              │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • Over-educating aware audience   │ • Mirrors prospect's thoughts              │
│ • Flat copy (no escalation)       │ • Builds stakes progressively              │
│ • Generic language                │ • Identification language                   │
│ • No emotional gradient           │ • Clear desire progression                  │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Awareness Level Targeting (25%) 2. Intensification Arc (25%)               │
│ 3. Identification Language (20%)    4. Educational Appropriateness (15%)      │
│ 5. Desire Progression (15%)                                                    │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 3. THE HALBERT — Offer Surgeon                                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: Gary Halbert (Direct Response Master)                                   │
│ Specialization: Offer structure, objection handling, urgency/scarcity           │
│                                                                                  │
│ OFFER FORMULA:                                                                  │
│ • What exactly do I get?                                                        │
│ • Why is it valuable to ME?                                                     │
│ • Why NOW?                                                                      │
│ • How do I overcome objections?                                                 │
│ • What's my next step?                                                          │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • "Great savings" (generic)       │ • "$47 off, ends Friday"                    │
│ • "Limited time" (vague)          │ • "Only for people over 50"                 │
│ • No clear CTA                    │ • "Risk-free 60-day guarantee"              │
│ • Unqualified offer               │ • One clear button/link                     │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Offer Clarity (25%)              2. Objection-Handling (20%)                │
│ 3. Urgency/Scarcity Credibility (20%) 4. Next Step Clarity (20%)              │
│ 5. Qualification & Targeting (15%)                                              │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 4. THE OGILVY — Brand-Response Bridge                                           │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: David Ogilvy (Ogilvy on Advertising, 1963)                             │
│ Specialization: Headlines, brand voice, research-backed claims                  │
│                                                                                  │
│ OGILVY'S CORE RULE: Headline = 80% of ad effectiveness                        │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • Weak or generic headline        │ • "7 reasons cardiologists recommend..."   │
│ • Claims without data             │ • "87% report less pain in 2 weeks"        │
│ • Hype & exaggeration             │ • Distinctive, memorable voice              │
│ • Generic brand voice             │ • Reads like trusted article                │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Headline Strength (25%)          2. Research-Backed Claims (20%)            │
│ 3. Body-Headline Alignment (20%)    4. Brand Voice Distinctiveness (20%)      │
│ 5. Intelligence & Respect (15%)                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 5. THE CAPLES — Headline Alchemist & Tester                                     │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: John Caples (Tested Advertising Methods, 1932)                         │
│ Specialization: Headline formulas, testing methodology                          │
│                                                                                  │
│ PROVEN HEADLINE FORMULAS:                                                       │
│ 1. HOW-TO → "How to lose 10 pounds in 30 days"                                │
│ 2. BENEFIT → "Save $47 on your heating bill"                                   │
│ 3. NEWS → "New breakthrough formula makes sleep easier"                       │
│ 4. QUESTION → "Are you making this common mistake?"                           │
│ 5. CURIOSITY → "The secret doctors don't want you to know"                    │
│ 6. SPECIFIC NUMBER → "7 ways to reduce anxiety naturally"                     │
│ 7. TESTIMONIAL → "How I tripled my income without a degree"                   │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • Generic headline                │ • Uses proven formula                       │
│ • Too clever/obscure              │ • Specific numbers (not round)              │
│ • No numbers/personalization      │ • Uses "you/your"                          │
│ • Weak benefit promise            │ • Compels reading                           │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Compulsion to Read (25%)        2. Headline Formula Strength (25%)          │
│ 3. Specificity & Numbers (20%)     4. Personalization (15%)                   │
│ 5. Clarity over Cleverness (15%)                                                │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 6. THE FOGG — Behavior Activation Engineer                                      │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: BJ Fogg (Tiny Habits, Behavior Model)                                   │
│ Specialization: Motivation, Ability, Prompt (MAP framework)                    │
│                                                                                  │
│ THE MAP FORMULA:                                                                │
│ BEHAVIOR = MOTIVATION + ABILITY + PROMPT                                       │
│            (emotional)   (ease)    (trigger)                                    │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • High motivation, high friction  │ • Clear benefit (motivation)                │
│ • Vague prompt or no prompt       │ • Single-click next step                    │
│ • Too much info to process        │ • Specific CTA ("Add to cart")             │
│ • Conflicting signals             │ • Aligned components                        │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Ability/Friction Reduction (30%) 2. Motivation Clarity (25%)               │
│ 3. Prompt Specificity (25%)        4. Friction Point Anticipation (10%)       │
│ 5. Component Alignment (10%)                                                    │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 7. THE CIALDINI — Persuasion Principle Validator                               │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: Robert Cialdini (Influence: Psychology of Persuasion)                 │
│ Specialization: 6 authentic persuasion principles                              │
│                                                                                  │
│ THE 6 PRINCIPLES:                                                               │
│ 1. RECIPROCITY → Give value first, obligation to return                       │
│ 2. COMMITMENT → Act consistent with stated positions                          │
│ 3. SOCIAL PROOF → Believe what others believe                                 │
│ 4. AUTHORITY → Trust experts and credible sources                             │
│ 5. LIKING → Influenced by those we like                                       │
│ 6. SCARCITY → Value things more when limited                                  │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • Fake authority                  │ • Multiple authentic principles             │
│ • Manufactured proof              │ • Real testimonials & credentials           │
│ • Single-principle reliance       │ • Real scarcity, real deadlines            │
│ • Manipulative tone               │ • Ethical, respectful approach              │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Social Proof Quality (20%)       2. Authority Legitimacy (20%)             │
│ 3. Reciprocity Authenticity (15%)   4. Scarcity Credibility (15%)            │
│ 5. Principle Diversity (15%)        6. Ethical Use/No Manipulation (15%)     │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 8. THE SUTHERLAND — Perceived Value Optimizer                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│ Author: Rory Sutherland (Alchemy & Behavioral Economics)                       │
│ Specialization: Perceived value, reframing, psychological positioning          │
│                                                                                  │
│ KEY INSIGHTS:                                                                   │
│ • Perceived value > objective value                                             │
│ • Small psychological changes >> big feature changes (at 1/10 cost)            │
│ • Reframe problem = reframe solution                                            │
│ • Status & identity matter more than specs                                      │
│ • Loss aversion (2:1) > gain desire                                            │
│                                                                                  │
│ RED FLAGS:                        │ GREEN FLAGS:                                │
│ • Feature-focused only            │ • Framed as loss-avoidance                  │
│ • No reframe or perspective       │ • Identity/status appeal                    │
│ • Generic story (or no story)     │ • Meaningful narrative                      │
│ • No identity/meaning appeal      │ • Psychological sophistication              │
│                                                                                  │
│ WEIGHT PRIORITIES:                                                              │
│ 1. Perceived Value Focus (25%)     2. Reframing Clarity (25%)                 │
│ 3. Loss Aversion Trigger (15%)     4. Identity & Status Appeal (15%)         │
│ 5. Narrative Over Specs (10%)      6. Psychological Sophistication (10%)     │
└─────────────────────────────────────────────────────────────────────────────────┘
